Web-GIS Spatial Analytics Dashboards & BI

Readiness Index (Growth Areas)

A simple, comparable index that scores each future urban growth area by coverage (network density) and capacity (pipe size), normalised to 0–1. Built offline from water-main geometry and published as a lightweight web map + ranked table for quick exploration.
